# Quadro dos heróis

O Quadro dos Heróis foi a conquista do ano de 2025 do Jumas do ramo de Seberi. 

O quadro busca trazer as principais personalidades heróicas do movimento, alinhadas em três eixos: Guerra/Fundação, Sacerdotes, Leigos (Pessoas "comuns"). A partir desses exemplos, espera-se fomentar nos membros a curiosidade sobre a vida dessas pessoas, e como buscaram viver os ideais de Schoenstatt e da Igreja.

Esse repositório contém um _preview_ do quadro, e os arquivos para o design no Figma com maior qualidade. O projeto no Figma também pode ser visualizado [aqui](https://www.figma.com/design/fepAr8XvNM1vlaOLlNTzuk/Quadro-dos-Her%C3%B3is?node-id=0-1&t=yUhq3Kc6OFyhkCVx-1).
